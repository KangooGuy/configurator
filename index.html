Beacon XL specifics:
Adv interval unit: 0.625 ms — e.g. 3200 = 2000 ms, 800 = 500 ms.

LoRa Configuration

LR1110 store-and-forward. Set the expected number of transmissions per message directly to model your network coverage (typical range: 1.0–3.0).
—
E[tx] = average number of LoRa transmissions per logical message (1.0 = perfect reception, 3.0 = max retries always needed).
Formula: E[tx] = 1 + (1−p₁) + (1−p₁)(1−p₂) — e.g. p₁=50%, p₂=75% → E[tx] = 1.625  ·  default 1.2 ≈ p₁=80%, p₂=100%
Confirmed (every N): TX + RX1 + RX2 + S&F retries  ·  Unconfirmed (others): TX only, no DL, no retry
Airtime from Semtech AN1200.13, EU868, BW=125 kHz, CR=4/5, 51+13 bytes, ITX=35 mA, IRX=10 mA

Tracking & Intervals

Moving and steady intervals combine with movement duration to derive total position fixes per day.
—
Fixes/day = movement_h × 3600 / moving_interval + (24 − movement_h) × 3600 / steady_interval
GNSS only occurs outdoors · WiFi scan covers indoor events
Tag S/L: autonomous GNSS only  ·  Tag XL / Lumen: assisted GNSS only  ·  Cellular: both modes

Energy Harvesting (PV Cell)

Physics-based model: lux → irradiance → PCE(irradiance) → power. Each technology has a distinct efficiency curve for indoor (LED/fluorescent) vs outdoor (AM1.5 solar) spectrum. PMIC efficiency η=0.90 applied.
—

Backup Battery (Non-rechargeable)

A non-rechargeable primary cell used as fallback when the main rechargeable battery is depleted. Its capacity is added to total available energy.
—
Self-discharge rates: CR2032 ≈ 1%/yr · L91 ≈ 0.3%/yr · Li-SOCl₂ ≈ 1%/yr · LiFePO₄ ≈ 1%/yr
Backup capacity is added to the main battery total for lifetime calculation. In practice the backup activates only after the rechargeable is exhausted.
